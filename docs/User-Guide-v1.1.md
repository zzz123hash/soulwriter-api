# SoulWriter 用户手册

> 版本：v1.1 | 更新：2026-04-11

## 简介

SoulWriter（灵魂创作者）是一款专为小说作者设计的创作工具，支持本地存储、云端同步（规划中）。

---

## 快速开始

### 1. 创建书本

1. 打开应用后，点击首页的「+ 创建新书」按钮
2. 输入书名、作者（可选）、简介（可选）
3. 点击「创建」

### 2. 打开书本

1. 在书架页面点击书本卡片上的「打开」按钮
2. 进入内页，可以看到左侧导航

### 3. 管理内容

- 📁 角色 - 故事中的人物
- 🎁 物品 - 装备、道具、关键物品
- 📍 地点 - 故事发生的场所
- 📖 章节 - 小说章节
- ✍️ 写作 - 创作界面（开发中）
- 🌳 创世树 - 剧情结构（开发中）

### 4. 返回书架

点击左侧顶部的「← 返回」按钮

## 功能说明

### 📚 Books 模块（核心）

| 功能 | 状态 | 说明 |
|------|------|------|
| 创建书本 | ✅ 已完成 | 输入书名、作者、简介 |
| 查看书架 | ✅ 已完成 | 卡片式展示所有书本 |
| 打开书本 | ✅ 已完成 | 点击进入内页 |
| 删除书本 | ✅ 已完成 | 确认后删除 |
| 导入书本 | ✅ 已完成 | 导入 .soul 或 .json 文件 |
| 导出书本 | 🔜 进行中 | 导出为标准格式 |

**存储方式：**
- 每个书本存储为独立的 `.soul` 文件
- 位置：`books_data/{bookId}.soul`

### 🔧 工具栏

右上角工具栏：
- 📋 日志 - 展开操作日志面板
- 📖 文档 - 查看使用文档
- ⚙️ 设置 - 应用设置

### 🌳 创世树

创世树是故事元素的可视化结构，用于：
- 规划故事核心主题
- 梳理角色关系
- 构建世界观

## 文件格式

### .soul 文件

SoulWriter 专用格式，实际上是 SQLite 数据库文件，包含：
- 书本信息
- 章节内容
- 记忆宫殿节点
- 向量索引（未来）

### 导入/导出

**支持格式：**
- `.soul` - SoulWriter原生格式（推荐）
- `.json` - 通用JSON格式（开发中）

## 常见问题

### Q: 为什么创建书本没反应？
1. 检查右下角日志面板是否有错误信息
2. 确保API服务正在运行
3. 刷新页面重试

### Q: 数据存储在哪里？
本地 `books_data` 目录，每个书本一个 `.soul` 文件

### Q: 如何备份数据？
直接复制 `books_data` 文件夹即可
