## Features

### Genesis Tree
- Visual worldbuilding with interactive tree structure
- Complete flow: Seed → Node → Relationship Tree
- Plot node types: Setup/Development/Turn/Resolution + Character/Setting
- Nvwa Engine integration - auto-create characters from genesis nodes

### Nvwa Engine
- AI-driven character simulation engine
- Quantum entanglement algorithm for character pair selection
- Tension control system (plot rhythm)
- Dynamic activation parameters
- Layered memory system (Buffer/Core/Recall/Archival)
- K-line recording for character growth tracking

### Book Management
- `.soul` format for final output
- `works/` for editing/preview
- Full support: Chapters, Roles, Items, Locations

### Theme System
- **Soft** (default) - Neumorphism + Frosted glass, Lavender + Mint
- **Dark** - Deep frosted glass, eye-protection mode
- **Auto** - Follows system preference

## Quick Start

```bash
git clone https://github.com/zzz123hash/soulwriter-api.git
cd soulwriter-api
npm install
npm start
# or
node src/app.js

# Visit
open http://localhost:3000/dashboard/
```
