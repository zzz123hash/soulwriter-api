# SoulWriter - 灵魂创作者

> 创世树 × 女娲引擎 AI 创作平台 | 世界架构设计师

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](https://github.com/zzz123hash/soulwriter-api/blob/main/LICENSE)
[![Node](https://img.shields.io/badge/node-%3E%3D18.0-green.svg)](https://nodejs.org/)

**SoulWriter** 不同于普通的 AI 写小说工具，是一个**设计完整世界**的工具。

## ✨ 核心功能

| 功能 | 说明 |
|------|------|
| 🌳 创世树 | 种子→节点→关系树，构建世界观结构 |
| 🔮 女娲引擎 | 分层记忆、量子纠缠选角、张力控制 |
| 🌍 语种翻译 | 文化智能适配（正在升级为AI原生） |
| 📚 书本管理 | 角色/物品/地点/事件全支持 |

## 🚀 快速开始

\`\`\`bash
git clone https://github.com/zzz123hash/soulwriter-api.git
cd soulwriter-api
npm install
npm start
\`\`\`

访问 http://localhost:3000/dashboard/

## ⚠️ 翻译系统说明

当前版本使用**规则+AI混合**模式，存在局限性。

**正在开发AI原生翻译**，目标：
- 语义理解而非关键词替换
- 场景感知（宫廷/江湖/现代）
- 情绪适配（悲伤场景不用圣诞）
- 人物背景（文人用典、武将用兵器）

详见 [docs/TRANSLATION.md](docs/TRANSLATION.md)
